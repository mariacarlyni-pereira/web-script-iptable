# 🖥️ IP Terminal Manager v2.0

Um gerenciador de endereços IP estilizado com interface de terminal (Retro Hacker Style). Este projeto permite realizar operações de CRUD (Create, Read, Update, Delete) em uma tabela de IPs, utilizando puramente HTML, CSS e JavaScript.

## 🚀 Funcionalidades

* **Listagem (Read):** Visualização de endereços IP, máscaras de sub-rede e versões (IPv4/v6) em uma tabela dinâmica.
* **Cadastro (Create):** Adição de novos IPs através de um formulário integrado.
* **Edição (Update):** Alteração de registros existentes diretamente via prompt.
* **Exclusão (Delete):** Remoção de IPs da lista com confirmação.
* **Interface Retro:** Design focado em contraste preto e verde neon para simular um terminal Linux antigo.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica dos elementos.
* **CSS3:** Estilização customizada com foco em usabilidade e estética hacker.
* **JavaScript (ES6+):** Lógica de manipulação do DOM e gerenciamento de estado do array.

## 📸 Demonstração Visual

A interface segue o padrão técnico exigido:
- **Header:** Linha de comando simulada `root@system:~$`.
- **Table:** Cabeçalho em destaque (verde sólido) e linhas separadas por bordas sutis.
